Supported Metric Profiles

Alcubierre Drive

The classic Alcubierre warp drive metric that contracts spacetime in front of the vessel and expands it behind.

$$ds^2 = -dt^2 + (1 - f(r,t))dr^2 + r^2d\theta^2 + r^2\sin^2\theta\,d\phi^2$$
Profile: $f(r,t) = \text{interpolated\_alcubierre}(r)$

Nat√°rio Drive

An alternative warp drive formulation that provides different energy requirements and stability properties.

$$ds^2 = -dt^2 + (1 - f(r,t))dr^2 + r^2d\theta^2 + r^2\sin^2\theta\,d\phi^2$$
Profile: $f(r,t) = \text{interpolated\_natario}(r)$

Coordinate System

All metrics use spherical coordinates $(t, r, \theta, \phi)$ with axial symmetry about the z-axis. The profile functions have compact support in the radial coordinate $r$.

Usage

1

Prerequisites

Install required dependencies:

pip install requests sympy

Ensure you have a LaTeX distribution with the amsmath package.

2

Clone Repository

git clone https://github.com/arcticoder/warp-bubble-metric-ansatz.git
cd warp-bubble-metric-ansatz
3

Generate Metrics

Run the generator script to create the LaTeX document:

python scripts/metric_ansatz.py
4

Compile LaTeX

pdflatex metric_ansatz.tex

This produces metric_ansatz.pdf with the formatted metric equations.
